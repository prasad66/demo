:link
The :link CSS pseudo-class represents an element that has not yet been visited. It matches every unvisited <a> or <area> element that has an href attribute.

Eg: The color of the link usually be blue, here the background and the text color has been changed to teal and white respectively as set with the css.

:visited
The :visited CSS pseudo-class represents links that the user has already visited. For privacy reasons, the styles that can be modified using this selector are very limited because of privacy issues.

Eg: When you click the link above the background and the text color will be changed to yellow and pink respectively as set with the css.

:hover
The :hover CSS pseudo-class matches when the user interacts with an element with a pointing device, but does not necessarily activate it. It is generally triggered when the user hovers over an element with the cursor (mouse pointer).

Eg: When hovered the background changes to blue and text color to black

:active
The :active CSS pseudo-class represents an element (such as a button) that is being activated by the user. When using a mouse, "activation" typically starts when the user presses down the primary mouse button.

Eg: When the link is active (i.e) the moment when it is clicked or holded up on, then the active pseudo class changes the text color to brown and the background to blue
